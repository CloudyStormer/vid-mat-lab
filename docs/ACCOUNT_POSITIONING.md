# 账号定位与表达规范

## 一句话定位

“内耗儿绝缘体”站在普通年轻打工人的一边，用简短、有依据、带一点轻松感的内容，拆解工作、生活、金钱和关系中的压力，帮助观众少一点自责和内耗。

## 内容主线

1. 工作压力：加班、绩效、职场沟通、情绪劳动、职业迷茫。
2. 生活与金钱：房租、消费焦虑、存钱、休息羞耻、生活成本。
3. 伴侣与关系：误解、沟通、边界、付出不对等、情绪支持。
4. 三类交叉主题：工作影响关系、金钱引发矛盾、压力侵占生活。
5. 轻入口：健身、吃饭、散步、通勤和爱好，用来承接“释放压力”，但不改变账号主线。

## 固定 IP

- 名称：白色小电阻人。
- 角色含义：外界不断施压，但它不再把所有压力都“吃进去”。
- 角色气质：有点疲惫、有点嘴硬、清醒但不刻薄，最后能找到一点松弛。

## 语言风格

目标是“自然开口、专业收束”。

推荐：

- “最近经常刷到……”
- “第一个。……”
- “严格来说，不能直接画等号。”
- “真正需要注意的是……”
- “所以，别死背公式。”

避免：

- “方向有些没错，问题是……”这类书面且拗口的开场。
- 一句话堆三四个解释，导致配音拖沓。
- 无条件的“必然、一定、绝对、全是”。
- 过度煽情、虚假共情、成功学和廉价鸡汤。
- 每集都用完全相同的口号强行结尾。

## 单段口播标准

- 一段只承担一个信息任务。
- 先报主题，再给结论，最多补一个必要原因。
- 一般控制在 25 至 50 个汉字；复杂健康内容可以稍长。
- “第一个、第二个”后保留自然停顿。
- 能删掉而不影响理解的词，优先删掉。

## 结尾原则

结尾应完成两件事：

1. 用一句话收束本集结论。
2. 自然连接压力或内耗。

示例：

~~~text
健身是释放压力，别让它成为新的内耗。
~~~

不是每集都必须出现“压力可以来，但我不吃”。它可以作为账号精神，而不是机械口号。
