# 零耗态·超导觉醒｜角色资产 v1

这是“内耗儿绝缘体”的科技觉醒形态。设计原则是保留约 70% 原 IP 识别度，只用约 30% 的装甲、材质和光效做升级，避免变成普通人形机甲。

## 固定识别特征

- 圆滚滚的白色电阻身体和短手短脚。
- 身体中部完整的灰色电阻腰带。
- 腰带正面的白色锯齿电阻纹。
- 蓝色电路纹与唯一一枚黄色闪电核心。
- 黑色或石墨色粗轮廓。
- 面部只有一张小黑嘴，位于眼睛下方、灰色腰带上方。
- 腰带和锯齿纹永远不是嘴，不得开合，不得出现牙齿或舌头。

## 视觉方向

- 风格：卡通轮廓与比例不写实，材质和光影采用电影级半写实 3D/PBR。
- 主色：珍珠白、石墨灰、电光蓝；黄色只作为闪电核心点缀。
- 场景：干净的深蓝黑未来能源舱，不堆砌赛博朋克元素。
- 禁止：第二张嘴、多余肢体、人类肌肉比例、红金机甲、武器、翅膀、已有影视机甲仿作、文字、Logo 和水印。

本目录属于跨集可复用品牌资产；具体一集的口播、视频提示词和剪辑参数仍保存在对应单集目录。
