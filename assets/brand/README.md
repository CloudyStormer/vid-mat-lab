# 品牌与 IP 资产

## 白色小电阻人

主参考图：[resistor-mascot-reference.png](resistor-mascot-reference.png)

它是“内耗儿绝缘体”的固定叙事角色。生成新图片时，参考图承担“角色身份参考”，不是普通风格参考。

必须保留：

- 圆滚滚的白色身体。
- 下半身浅灰色电阻腰带和锯齿电阻纹。
- 蓝色电路纹和蓝色电流符号。
- 头顶黄色闪电。
- 黑色粗轮廓、短手短脚、淡粉脸颊。

可以改变：

- 动作、手持物和轻微表情。
- 服装或场景中的少量必要配件。
- 光线、视角和背景氛围。

不要改变：

- 物种、主体颜色、身体比例和核心电阻结构。
- 不要变成猫、猪、人类、鸡蛋或普通机器人。
- 不要让 AI 在图片中生成标题、Logo、水印或乱码。

注意：原参考图右下角包含原生成平台标记。正式发布图片应使用无水印的新生成画面，不直接把参考图作为作品画面。
