# 品牌与 IP 资产

## 白色小电阻人

主参考图：[resistor-mascot-reference.png](resistor-mascot-reference.png)

它是“内耗儿绝缘体”的固定叙事角色。生成新图片时，参考图承担“角色身份参考”，不是普通风格参考。

必须保留：

- 圆滚滚的白色身体。
- 下半身浅灰色电阻腰带和锯齿电阻纹。
- 蓝色电路纹和蓝色电流符号。
- 头顶黄色闪电。
- 黑色粗轮廓、短手短脚、淡粉脸颊。

## 角色拓扑锁定（所有图片与视频提示词必须重复）

- 角色只有一个连续、完整、不可拆分的白色蛋形机体，不是“头加身体”，也不是上下两颗蛋叠在一起。
- 脸、眉毛、眼睛、鼻子和唯一小嘴位于同一颗蛋的上半部分；同一颗蛋的下半部分就是身体和电阻区域。
- 浅灰色电阻腰带只是环绕同一颗蛋中下部的一条固定结构带。禁止第二条腰带、腰带下方的第二颗蛋、独立腹部、胯部圆球、圆筒骨盆和第二躯干。
- 全程恰好一双胳膊、一双手、一双腿、一双脚。左右肢体始终连接在同一颗蛋对应的一侧，动作转换不得用残影或临时新肢体补帧。
- 左脚位于机体左下方，右脚位于机体右下方；站立或落地时双脚必须处于同一支撑平面，禁止错层、交叉、漂浮、合并、左右互换和额外鞋子。
- 头顶只能保留黄色扁平闪电折线和蓝色扁平电流折线，禁止把任何折线实体化成圆球、第二颗头、小机器人或立体天线。
- 宁可降低动作幅度，也不能破坏上述主体和四肢拓扑。每一个独立生成提示词都要完整重复这些约束，不能只放在文档开头作为共用说明。

可以改变：

- 动作、手持物和轻微表情。
- 服装或场景中的少量必要配件。
- 光线、视角和背景氛围。

不要改变：

- 物种、主体颜色、身体比例和核心电阻结构。
- 不要变成猫、猪、人类、鸡蛋或普通机器人。
- 不要让 AI 在图片中生成标题、Logo、水印或乱码。
