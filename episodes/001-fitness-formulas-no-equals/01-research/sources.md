# 第 001 集研究与科学依据

## 待核验的原始说法

1. 有氧 + 不控制饮食 = 变胖
2. 有氧 + 控制饮食 = 体重下降
3. 力量 + 不控制饮食 = 变壮
4. 力量 + 有氧 + 控制饮食 = 皮质醇飙升
5. 力量 + 控制饮食 = 体脂率降低

## 本集采用的结论边界

### 1. 有氧加不控制饮食

不能推出必然变胖。长期体重趋势取决于能量摄入与消耗；运动后额外进食可能抵消运动消耗。

### 2. 有氧加控制饮食

在形成合理、可持续能量缺口的前提下，更可能减重减脂。“控制饮食”不能表达为不吃饭或极端节食。

### 3. 力量训练加不控制饮食

不能推出自动变壮。肌肉增长需要训练刺激、营养、恢复和时间；长期能量过剩也可能同时增加脂肪。

### 4. 力量、有氧加控制饮食

不能推出皮质醇必然“飙升”。运动后的短期升高可以是正常生理反应；长期低能量可用性、过度训练和睡眠不足更值得关注。

### 5. 力量训练加合理饮食

通常更有利于减脂期间保留肌肉并改善身体成分，但结果仍受总能量、训练质量、恢复和个体差异影响。

## 表达限制

- 不提供个体化医疗诊断。
- 不把短期体重变化等同于脂肪变化。
- 不把皮质醇本身描述成有害物质。
- 不承诺任何训练方式必然减重、增肌或改善体脂。
