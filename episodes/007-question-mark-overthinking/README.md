## 本集目标

- 面向谁：容易从简短回复、语气和标点中揣测他人态度的年轻打工人。
- 解决什么问题：把对方发来的有限信息，自动扩写成否定、针对或不耐烦，继而反复自责和内耗。
- 观众看完得到什么：学会区分“已经发生的事实”和“自己补出的剧情”，并用一句澄清代替长时间猜测。
- 与账号主线的关系：回到年轻打工人的职场压力和心理内耗；相比第 003 集的外部边界，本集重点处理内部脑补。

## 核心判断

对方发来的只是一个问号；不耐烦、否定和针对，可能都是我们在信息不完整时自动补上的剧情。少猜一点，多确认一句。

## 内容结构

- 形式：静态配图 + 口播 + 字幕，不制作角色动画。
- 目标时长：55—65 秒。
- 画面数量：8 张，一图对应一段独立口播。
- 开场：同事只回复一个问号，IP 立刻开始脑补。
- 中段：把“事实”和“解释”分开，指出大脑会在不确定时自动补剧情。
- 行动：直接询问“是哪个地方没说清楚？我补充一下。”
- 结尾：不要把一个问号扩写成一整部职场悬疑片。
