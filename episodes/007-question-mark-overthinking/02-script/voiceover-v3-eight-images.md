# 最终口播候选 v3｜八段对应八张图

每个代码块对应一张图片和一段独立配音。正常语速加自然停顿，预计总时长约 55—62 秒。

## 01｜问号出现

```text
同事只回了一个问号。你是不是盯着屏幕，整个人都僵住了？
```

对应图片：`03-visuals/generated-v1/01-question-mark-freeze.png`

## 02｜脑补开始

```text
他是不耐烦？在质疑我？还是准备甩锅？先停一下。事实只有一个：他没看懂，或者需要确认；其他态度，都还没有证据。
```

对应图片：`03-visuals/generated-v1/02-three-imagined-scenarios.png`

## 03｜第一次澄清

```text
先直接问：你是没看懂哪一部分，还是对结论有疑问？
```

对应图片：`03-visuals/generated-v1/03-ask-specific-question.png`

## 04｜没回复就补齐信息

```text
如果没回，就把背景、要他确认的事和时间点说清楚：这个决定会影响今天的排期，麻烦三点前确认。
```

对应图片：`03-visuals/generated-v1/04-context-choice-deadline.png`

## 05｜按紧急程度换方式

```text
事情紧急，就打电话或者当面说；不紧急，就给对方合理的回复时间。
```

对应图片：`03-visuals/generated-v1/05-urgent-vs-wait.png`

## 06｜到期留痕

```text
到点还没回，可以留痕：为了不影响排期，我先按 A 方案推进；有调整请告诉我。
```

对应图片：`03-visuals/generated-v1/06-record-and-proceed.png`

## 07｜授权边界

```text
但付款、发布、合同这些需要明确授权的事，不能把沉默当成同意，要继续确认或者找负责人。
```

对应图片：`03-visuals/generated-v1/07-authorization-boundary.png`

## 08｜收尾

```text
职场沟通别靠猜：问题问具体，期限说清楚，结果留记录。一个问号，不值得你脑补一整晚。
```

对应图片：`03-visuals/generated-v1/08-turn-off-inner-movie.png`
