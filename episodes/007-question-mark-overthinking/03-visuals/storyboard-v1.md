# 八张静态图分镜 v1

统一要求：横版 16:9、固定小电阻人 IP、图片内不生成文字、统一轻电影感卡通风格。问号、聊天内容和“事实 / 脑补”等文字后期在剪映添加。

| 图号 | 对应内容 | 画面动作 | 建议时长 |
|---:|---|---|---:|
| 01 | 同事只回一个问号 | 夜晚办公室，小电阻人坐在电脑前，手机亮起；它突然僵住，画面留出聊天框位置 | 5—6 秒 |
| 02 | 脑子马上开会 | 小电阻人头顶出现三块半透明想象画面，分别表现不耐烦、质疑和甩锅；现实办公区保持安静 | 7—8 秒 |
| 03 | 在心里吵了三轮 | 桌面堆满被删改的消息草稿，小电阻人来回检查，疲惫焦躁 | 6—7 秒 |
| 04 | 事实和脑补分开 | 画面左右分区：左边只有手机和一个简单问号，右边是夸张的职场悬疑想象；小电阻人站中间观察 | 7—8 秒 |
| 05 | 可能只是没看懂 | 同事在另一张办公桌前看着附件和流程图，露出普通困惑表情，没有愤怒或嘲讽 | 6—7 秒 |
| 06 | 直接问清楚 | 小电阻人恢复平静，坐直身体，认真输入回复；构图给字幕和聊天框留白 | 6—7 秒 |
| 07 | 沟通恢复正常 | 两人在同一张流程图前确认具体位置，小电阻人补充信息，对方自然点头 | 6—7 秒 |
| 08 | 别扩写成悬疑片 | 小电阻人合上一本夸张的“悬疑剧本”道具，桌上只剩手机；神情放松，画面右侧留结尾字幕区 | 7—8 秒 |

## 视觉节奏

1. 01—03 使用偏冷的夜间办公光，表现不确定和反复脑补。
2. 04 是认知转折，画面结构必须一眼看出“一个事实”和“一大堆想象”的反差。
3. 05—08 逐渐转为中性、温暖的光线，表现问题变得具体、可处理。
4. 静态图只做轻微 100%—106% 缩放；转场统一 0.2—0.3 秒淡化。
