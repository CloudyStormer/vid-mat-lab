# 八张图片生成提示词 v1

## 共同规格

```text
用途：第 007 集横版静态配图。
画幅：16:9。
场景：现代开放式办公室夜晚，冷蓝环境光与暖色台灯形成电影感反差。
风格：精致电影感 3D 卡通动画静帧，轻松、有故事性，不做 PPT 或信息图。
角色：严格参考 assets/brand/resistor-mascot-reference.png。
必须保留：一个连续的白色圆形身体；灰色电阻腰带；一条白色锯齿电阻纹；蓝色电路纹；粗黑轮廓；头顶黄色闪电和扁平蓝色脉冲符号；两只眼睛；腰带上方唯一一张嘴；两只手和两只脚。
禁止：头顶球体、圆球、额外机器人头、额外角色身体、多手、多嘴、多眼、把腰带当嘴、乱码、水印、Logo、PPT 排版。
```

## 逐张场景

### 01｜问号出现

```text
夜间办公室里，小电阻人坐在电脑前举着手机。手机聊天气泡中只有一个“?”。它盯着屏幕瞬间僵住，眼睛略微睁大，手机冷光照亮面部。角色放在画面右中部，左侧留字幕空间。
```

### 02｜三种脑补

```text
延续上一张办公室和机位。小电阻人不安地盯着手机。头顶出现三个半透明紫灰色想象气泡：同一名年轻同事抱臂不耐烦；同事皱眉审视报告；同事把夸张文件堆推过来。真实办公室安静无人，三组画面必须明显属于脑补。
```

### 03｜直接澄清

```text
同一办公室。小电阻人把手机立在电脑旁，用两只手平静打字；屏幕只有空白聊天气泡和放大镜符号。上一张的三个紫色脑补气泡正在消散，角色姿态恢复专注。
```

### 04｜背景、动作和时间点

```text
同一办公室。小电阻人一手放在键盘，一手指向时钟。桌面上方出现三个融入场景的蓝色全息道具：带回形针的项目文件夹；圆形与三角形两张选择卡；指向三点的时钟。三组道具汇入一条聊天气泡。
```

### 05｜紧急或等待

```text
用办公室建筑和光线自然形成左右平行情境。左侧红色提醒灯亮起，小电阻人打电话处理紧急事项；右侧冷蓝环境中，小电阻人把手机扣在桌面，继续专注工作并等待普通事项回复。每侧角色都保持正确 IP 特征。
```

### 06｜到期留痕并推进

```text
同一办公室，时间已到。小电阻人右手把蓝色棋子放到项目板的蓝色圆点路径，左手点击平板；平板只显示聊天气泡和归档勾选图标。表情冷静负责，不得意、不赌气。
```

### 07｜授权边界

```text
同一办公室审批区域。发光红色暂停屏障后放置三样东西：带硬币的支付卡、带封印的合同、带向上箭头的发布屏幕。小电阻人双手拿空白审批文件，转向一名平静负责人继续确认，所有关键操作都尚未执行。
```

### 08｜关闭脑内悬疑片

```text
清晨暖光进入办公室。左侧紫色老式放映机正在关闭，墙上“不耐烦、质疑、甩锅”三个脑补投影逐渐消散。小电阻人左手按下关闭键，右手拿记录本。右侧桌面整齐摆放三样工具：聊天气泡上的放大镜、时钟、带勾的记录本。
```
