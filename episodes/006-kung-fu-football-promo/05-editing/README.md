# 剪辑与验收

## 先生成动画

1. 打开即梦或可灵的“首尾帧生视频”。
2. 首帧上传 `03-visuals/01-stadium-broadcast-start.png`。
3. 尾帧上传 `03-visuals/02-stadium-broadcast-end.png`。
4. 选择 9:16、10 秒、固定镜头或最低运镜强度。
5. 粘贴 `04-prompts/video-prompt-v4-four-locks-natural-football.md` 的主提示词；有单独负面框时同时粘贴其中的负面提示词。
6. 关闭自动配音和自动字幕；不要启用平台数字人口型，由 v4 提示词直接生成小电阻唯一嘴的拟说话开合。
7. 生成至少两个候选。若平台最高只支持 5 秒，按提示词中的两段方案拆分。

## 候选视频验收

- 角色全程只有一个、只有一张嘴。
- 嘴始终位于眼睛与灰色腰带之间。
- 灰色腰带不张嘴，白色锯齿纹不变成牙齿。
- 扩音喇叭只有一个，不穿过手和脸。
- 角色不分身、不多手多脚。
- 镜头没有突然推近、转场或背景重绘。
- 背景球员没有突然受惊、集体回头或靠近镜头。
- 足球只有一个，火焰没有变成爆炸。
- 卡通化星爷始终在左侧同一位置，银发、帽子、眼镜、衣服和交叠双臂不漂移。
- 星爷全程闭嘴，只允许自然眨眼和极轻微眼神变化，不生成台词或口型。
- 小电阻只有一个圆形头身一体主体，头顶蓝色折线始终是二维符号，不出现额外球形脑袋。
- 银灰色喇叭从第 0 帧就握在双手中，并持续存在到末帧，没有消失或突然出现。
- 小电阻口播期间唯一小嘴在腰带上方清楚开合，腰带和锯齿纹保持刚性。
- 小电阻全程恰好两只手：左手托喇叭筒身、右手握手柄，不出现额外手臂和手势。
- 远景球员连续带球、跑位、传球和射门，脚落地、球沿草坪移动，没有漂浮飞踢或惊吓反应。

## v2 失败记录

问题截图：`issues/v2-extra-head-missing-megaphone.png`。

- 蓝色二维电流符号被模型解释成一颗长有眼睛的球形小脑袋。
- 主角色被错误改成“独立头＋身体”的机器人结构，违反头身一体 IP 设定。
- 首帧中原本已有的银灰色喇叭消失。
- 处理：v2 停止使用，改用 v3 的拓扑、贴花、道具持久性和口型硬约束。

## 剪映时间线

1. 导入生成的 10 秒无声视频。
2. 生成或录制四句口播，建议 1.10—1.15 倍速，按 0、2.0、4.0、7.4 秒依次对齐。
3. 自动识别字幕后按 `02-script/voiceover-v1.md` 校对，不让“功夫女足”识别成同音字。
4. 字幕放在画面下方安全区，避开角色和喇叭；白字、黑色描边，不用花字动画。
5. 选轻热血、带鼓点但不喧闹的音乐。人声峰值约 -6 dB，音乐比人声低 14—18 dB。
6. 9.5 秒开始让音乐和广播波一起收束，尾帧至少停 0.3 秒。
7. 封面选择尾帧，顶部添加两行标题：“只能帮你到这了 / 宣传到这了”。

## 导出

- 画幅：1080×1920
- 帧率：30 fps
- 编码：H.264
- 码率：建议 12—16 Mbps 或剪映“更清晰”
- 音频：AAC，48 kHz
- 文件名建议：`kung-fu-football-promo-v1.mp4`
