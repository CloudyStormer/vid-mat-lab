# 黑幕揭场｜首尾帧跳上木台与防穿帮版 v7

本版只用于截图中的这一段：

- 左图：首帧，幕布已经拉开，小电阻仍在前景地面；
- 右图：尾帧，小电阻已经跳到后方木台上并站稳；
- 动作过程：完成拉幕 → 松开幕布 → 整个蛋体短距离跳向后方木台 → 双脚同时落台；
- 生成时长：5 秒；
- 画幅：9:16；
- 模式：首尾帧图生视频。

## 可直接复制的提示词

```text

使用我上传的两张图片生成一段5秒、9:16竖屏的首尾帧视频。

第一张图片是强制首帧，必须完整作为第0帧。
第二张图片是强制尾帧，必须完整作为最后一帧。
视频只能表现第一张图到第二张图之间的连续动作，最后准确停在第二张图。禁止把两张图片混合重绘，禁止自行创造第三个构图，禁止切镜头、转机位、变焦、闪现或瞬移。

【唯一正确的故事过程】
第一张图中，小电阻刚刚把黑色幕布拉开，仍站在前景地面。
第二张图中的木台原本就在球场后方，不是中途突然出现的新道具。
接下来，小电阻先完成最后一点拉幕动作，再松开幕布；然后保持原来的圆形蛋状身体和极短腿，以整个身体为一个刚性整体，沿一条短而清楚的向后、向上弧线，跳到后方木台上；最后双脚同时落在木台顶面并站稳，准确进入第二张尾帧。

这段不是走过去，不是跑过去，也不是伸长腿跨过去。必须看得出角色从前景跳向后方木台，但跳跃幅度要小、动作要轻。

【角色外形绝对冻结｜最高优先级】
全程只能是参考图中的同一个白色小电阻人：
只有一颗完整连续的圆形蛋状机体；
没有独立头部、没有脖子、没有第二颗蛋；
身体宽高比、身体大小、灰色电阻腰带高度、白色锯齿纹、蓝色电路纹、五官位置、黑色描边、黄色闪电、蓝色扁平脉冲符号全部保持不变。

黄色闪电和蓝色脉冲只是头顶的扁平符号。禁止出现球体、悬浮圆球、天线球、机器人小头、第二颗头或额外角色。

【腿绝对不能变长】
小电阻始终只有两条非常短、非常粗的卡通短腿和两只圆头短脚。
腿长必须逐帧保持与两张参考图完全一致，绝对不能变长。
禁止生成大腿、膝盖、小腿、脚踝、长腿、人类腿、长距离跨步、助跑、高抬腿、蹬腿和伸腿落地。

跳跃只能由整个蛋形机体整体移动完成，不能靠腿伸缩完成。跳起、飞行和落地过程中，两只短腿和两只脚始终紧贴蛋体底部，四肢比例完全不变。

【手与喇叭连续性】
全程只有两条胳膊和两只手，禁止第三只手、残影手、漂浮手、分叉胳膊和多余手指。

全程只有同一个银灰色喇叭。右手从第0帧到最后一帧始终握住同一个喇叭手柄，喇叭不能消失、复制、换手、变形或突然改变大小。

0—1.0秒：左手保持第一张图的抓幕姿势，完成最后一点拉幕动作。
1.0—1.8秒：原来的同一只左手松开幕布，抓幕位置的手立即消失；同一只左手沿连续短路径移向喇叭筒身。禁止同时出现抓幕手和托喇叭手。
1.8—5.0秒：左手托住喇叭筒身，右手握住喇叭手柄。两只手与喇叭保持固定接触，一起随整个蛋体移动并落到木台上。

【动作时间线】
0—1.0秒：幕布继续向画面两侧自然滑开，小电阻保持前景位置。
1.0—1.8秒：小电阻松开幕布，两手重新稳定抱住同一个喇叭。
1.8—3.4秒：整个蛋体沿短弧线向后方木台整体移动。角色因为进入画面纵深可以轻微自然缩小，但身体比例、腿长、手脚大小不能变化。
3.4—4.5秒：整个蛋体缓慢垂直下降，两只短脚同时接触木台顶面。
4.5—5.0秒：身体只做一次极轻的整体上下回弹，随后完全站稳并准确停在第二张尾帧。

【绝对禁止穿帮】
黑色幕布只能位于角色两侧或后方，不能穿过小电阻身体、手臂、喇叭、星爷或木台。
左手松开幕布以后，幕布不能继续黏在手上，也不能穿过手掌。
喇叭不能穿过角色的脸、身体、腰带、手臂或木台。
两只手不能穿过喇叭，手指不能忽然出现在喇叭另一侧。
两只脚不能穿过木台、陷入木台、漂浮在木台上方或落在不同高度。
木台不能突然移动、变形、放大、消失或穿过角色。
角色不能穿过幕布、星爷、木台或地面。
星爷不能与幕布、角色或木台相互穿透。
禁止任何肢体、道具、背景人物出现重叠错误、瞬间换位、闪烁、断裂、透明、复制或消失。

【背景连续性】
金色夕阳球场、左侧卡通化星爷、后方木台、灯光、草地和观众席必须连续稳定。
星爷保持左侧原位，闭嘴、双臂交叠、安静观看，不突然转身、不后退、不挥手、不受惊。
远景球员只做自然、轻微的跑位，不抢镜、不集体看向角色。

【嘴部】
唯一一张小嘴始终位于两只眼睛下方、灰色腰带上方。
本段尚未开始正式广播，小嘴保持闭合或只有一次极轻呼吸动作。
灰色腰带和白色锯齿纹是固定结构，不能开合、不能长牙、不能变成第二张嘴。

无文字、无字幕、无Logo、无水印。

最终优先级：
1. 两张图片必须分别成为首帧和尾帧；
2. 过程必须是拉开幕布后跳到后方木台；
3. 腿绝对不能变长；
4. 始终只有一颗蛋、两只手、两条短腿、两只脚和一个喇叭；
5. 任何角色、肢体、幕布、喇叭和木台都不能发生穿帮或穿透。

如果模型无法同时完成复杂跳跃和结构稳定，宁可把跳跃缩小成整个蛋体沿短弧线轻轻漂移，也绝对不能拉长腿、增加肢体或出现穿帮。
```

## 使用说明

1. 左图放在首帧槽位，右图放在尾帧槽位。
2. 不要把两张图都当作普通“参考图”上传。
3. 选择支持首尾帧的生成模式；固定机位、关闭强运镜。
4. 结构稳定优先于动作幅度，生成失败时先降低运动强度，不删除短腿和防穿帮约束。
