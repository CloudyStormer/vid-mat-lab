# 黑幕揭场｜三项素材压缩为后两段 v6

## 素材关系

基于用户当前确认的三项素材：

1. 已生成的 6 秒黑幕揭场视频；
2. `03-visuals/01-stadium-broadcast-start.png`；
3. `03-visuals/02-stadium-broadcast-end.png`。

前一段黑幕视频保留。本版只生成后面两段：

- 第二段：黑幕视频最后一帧 → 站上木箱准备宣布；
- 第三段：站上木箱准备宣布 → 举起喇叭正式广播。

不要再单独生成“拉幕”和“跳向木箱”两个中间视频。

## 第二段｜松开幕布，落到木箱

### 上传素材

1. 把现有 6 秒黑幕视频的最后一帧导出为首帧图片；
2. 把 `03-visuals/01-stadium-broadcast-start.png` 作为尾帧图片；
3. 生成时长：5 秒；
4. 模式：首尾帧图生视频、9:16、固定机位。

### 可直接复制的提示词

```text
使用我上传的两张图片生成一段5秒、9:16竖屏、固定机位的首尾帧视频。

第一张图片必须作为第0帧，第二张图片必须作为最后一帧。画面从第一张自然连续地过渡到第二张，最后准确停在第二张图片的构图和姿势。禁止切镜头、变焦、闪现、瞬移、重绘角色或更换场景。

【角色外形绝对锁定｜最高优先级】
全程是参考图中的同一个白色小电阻人。它只有一颗完整连续的圆形蛋状机体，没有独立头部、没有脖子、没有第二颗蛋。蛋体宽高比、身体大小、灰色电阻腰带高度、白色锯齿纹、蓝色电路纹、五官位置、黑色粗描边、黄色闪电和蓝色扁平脉冲符号必须保持不变。

头顶黄色闪电和蓝色脉冲只是扁平符号。头顶及其周围禁止出现球体、圆球、悬浮小头、机器人头、天线球和额外角色。

小电阻全程只有两条短胳膊、两只手、两条极短粗腿和两只脚。禁止第三只手、额外胳膊、残影手、分叉手、漂浮手、第三条腿和额外鞋子。腿必须保持参考图中的极短比例，禁止生成大腿、膝盖、小腿、脚踝、长腿、助跑、跨步和人类式跳跃。

全程只有同一个银灰色喇叭。右手从第0帧到最后一帧始终握住同一个喇叭手柄，喇叭不能消失、不能复制、不能突然换手。

【动作时间线】
0—1.2秒：延续第一张图。小电阻保持原位置，右手继续握住喇叭；原来的左手完成最后一点拉幕动作。黑色幕布自然滑向画面边缘。

1.2—2.0秒：原来的同一只左手完全松开幕布，并沿一条连续的短路径移向喇叭。左手离开幕布以后，原抓幕位置必须立刻消失，不能留下第二只手或残影手。

2.0—3.2秒：左右两手共同抱稳同一个喇叭。整个蛋形机体作为一个刚性整体，向木箱方向做非常短、非常轻的整体上移和前移。四肢长度完全不变，双脚始终贴近蛋体底部。不要奔跑，不要迈步，不要拉长腿。

3.2—4.4秒：小电阻作为一个完整蛋体缓慢垂直下降到木箱上。左右脚大小相同、方向相同，同时接触同一个水平箱面，鞋底同高。禁止一脚先落、前后错层、交叉、穿箱、悬空或多出第三只脚。

4.4—5.0秒：角色轻微整体回弹后站稳，准确进入第二张参考图的姿势。两手仍抱着同一个喇叭，唯一小嘴闭合，目光看向观众，准备开始宣布。

【背景锁定】
金色夕阳球场、木箱和左侧卡通化星爷保持与参考图一致。星爷始终闭嘴、双臂交叠、安静观看，不突然转身、不受惊、不说话。远景球员只做自然的小幅跑位和传球，不能集体停下或看向镜头。

【嘴部锁定】
小电阻唯一一张小嘴位于两只眼睛下方、灰色腰带上方。腰带和白色锯齿电阻纹永远是固定结构，不能开合、不能长牙、不能变成第二张嘴。本段尚未正式广播，嘴保持闭合或只做一次极轻呼吸动作。

无文字、无字幕、无Logo、无水印。若动作与角色结构冲突，宁可缩小跳跃、改为轻微整体平移，也不能增加手脚、拉长腿或改变IP外形。
```

## 第三段｜举起喇叭，正式广播

### 上传素材

1. `03-visuals/01-stadium-broadcast-start.png` 作为首帧；
2. `03-visuals/02-stadium-broadcast-end.png` 作为尾帧；
3. 生成时长：6 秒；
4. 模式：首尾帧图生视频、9:16、固定机位。

### 可直接复制的提示词

```text
使用我上传的两张图片生成一段6秒、9:16竖屏、固定机位的首尾帧视频。

第一张图片必须作为第0帧，第二张图片必须作为最后一帧。画面从第一张自然连续地过渡到第二张，最后准确停在第二张图片的构图、角色位置、喇叭角度、蓝色声波和背景状态。禁止切镜头、推拉镜头、闪现、瞬移、重绘角色或更换场景。

【角色与道具绝对锁定｜最高优先级】
全程只能出现参考图中的同一个白色小电阻人：一颗完整连续的圆形蛋状机体、一圈灰色电阻腰带、一条白色锯齿电阻纹、蓝色电路纹、黑色粗描边、黄色闪电和蓝色扁平脉冲符号。身体大小、宽高比、五官位置、腰带高度、四肢长度、颜色和材质不得变化。

头顶黄色闪电和蓝色脉冲只是扁平图形。禁止把它们变成球体、悬浮小头、第二颗机器人脑袋或立体角色。

小电阻全程只有两条胳膊、两只手、两条极短腿和两只脚。禁止第三只手、额外胳膊、残影手、漂浮手、第三条腿、长腿、大腿、膝盖和额外鞋子。

全程只有一个银灰色喇叭。喇叭从第0帧到最后一帧持续存在，不得消失、复制、变形、换颜色或突然出现在其他位置。右手始终握住喇叭手柄；左手始终托住喇叭筒身。若某只手被喇叭遮挡，就保持自然遮挡，绝不能在喇叭后面补画新手。

【动作时间线】
0—1.0秒：小电阻已经站稳在木箱上，保持第一张图的准备姿势。星爷在左侧安静站立，背景球员自然移动。

1.0—2.2秒：小电阻用两只原有的手共同抬起同一个喇叭。喇叭和双手作为一个连接牢固的整体，缓慢转向画面右前方。手不能松开、换位、复制或穿过喇叭。

2.2—4.8秒：喇叭到达广播位置后保持稳定。小电阻开始讲话。唯一小嘴在眼睛下方、腰带上方自然小幅开合，发音节奏清楚但动作克制；嘴不能移动到灰色腰带或锯齿纹上。灰色腰带和锯齿纹全程固定不动，永远不能开合或长牙。

讲话时，蓝色电路纹从暗到亮逐渐发光。喇叭口前方出现由小到大的蓝金色圆形声波，声波必须从喇叭口发出，不能从角色身体、嘴或天空中凭空出现。声波逐渐扩散，不突然闪现。

4.0—5.5秒：远景球员保持真实足球动作，一名球员自然助跑并完成一次射门；足球沿连续、合理的抛物线飞向画面右上方。足球只能有一个，不能复制、停在空中或突然改变方向。其他球员自然跑位，不集体看镜头、不受惊。

5.5—6.0秒：准确停在第二张参考图。小电阻站稳木箱、喇叭保持广播角度、蓝色声波完整可见、足球位于目标位置。唯一小嘴完成最后一个音节后轻微闭合。

【星爷与背景锁定】
左侧卡通化星爷全程保持原位置、原比例和原服装，闭嘴、双臂交叠、安静观看。他不说话、不挥手、不突然转身、不后退、不受惊。球场机位、金色夕阳、灯光、木箱和观众席保持一致。

【最终核验】
每一帧必须是：一个小电阻、一颗蛋状机体、一圈腰带、两只眼睛、唯一一张小嘴、两条短胳膊、两只手、两条极短腿、两只脚和一个喇叭。禁止头顶球体、第二颗头、第三只手、第二张嘴、腰带开口、喇叭消失和足球复制。

无文字、无字幕、无Logo、无水印。若复杂动作导致角色漂移，宁可减少球员动作和声波数量，也必须优先保持小电阻、双手、嘴和喇叭完全正确。
```

## 生成与剪辑顺序

1. 保留现有 6 秒黑幕揭场视频。
2. 从该视频导出最后一帧，和 `01-stadium-broadcast-start.png` 一起生成第二段。
3. 使用 `01-stadium-broadcast-start.png` 与 `02-stadium-broadcast-end.png` 生成第三段。
4. 剪辑顺序为：现有黑幕视频 → 第二段落箱 → 第三段广播。
5. 每段先检查头顶无球、恰好两只手、短腿未拉长、喇叭持续存在；结构正确后再检查动作节奏。
