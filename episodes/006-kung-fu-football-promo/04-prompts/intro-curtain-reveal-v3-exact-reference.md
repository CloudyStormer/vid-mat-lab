# 黑幕撕开揭场｜四段首尾帧视频提示词 v3

本版针对 v2 实测出现的第三只手和人类式长腿。核心变化：

1. 上传首尾帧与固定 IP 参考图是绝对角色模型，不允许视频模型重新设计角色。
2. 明确锁定参考图中的短臂、短腿和脚贴蛋体比例，禁止大腿、膝盖、小腿、跨步与人类跑姿。
3. 同一只手换任务时分成“结束旧动作—短暂停顿—同一只手开始新动作”，前后手势不能重叠。
4. 所有移动优先让整个蛋形角色轻微平移，不通过拉长四肢完成。

> 当前 `01-curtain-slit-emerge.png` 与 `02-stop-important-message.png` 中，左手分别承担“抓幕”和“停止手势”。这类首尾帧冲突仍是多手高风险来源。若 v3 仍出现第三只手，应重做第一张图，让左手从首帧起就是停止手势，或者把第一段拆成两段。

【上传图片外形绝对锁定｜最高优先级】
每一帧的小电阻人都必须与上传首帧、尾帧以及IP参考图完全一致。逐帧复制原图中的蛋体外轮廓、蛋体宽高比、身体大小、腰带高度和宽度、锯齿纹数量、五官位置、黄色闪电、蓝色折线、电路纹位置、黑色描边、颜色、材质、喇叭造型、手脚大小和四肢长度。禁止重新建模、重新设计、换画风、瘦身、增高、拉长、缩小头身比例或把角色改成人形。角色在视频中应当像“原图本身轻微动起来”，不能像重新生成了另一个角色。

【单一蛋体与固定四肢】
角色全程只有1颗完整连续的白色蛋形机体，禁止上下叠蛋、第二颗蛋、独立头部、脖子、第二躯干和腰带下方额外腹部。灰色电阻腰带只能有1圈，并且贴在同一颗蛋体表面。
角色全程恰好只有2条胳膊、2只手、2条腿、2只脚。禁止第三只手、额外胳膊、残影手、悬空手、道具后面补画的手、分叉肢体、第三条腿和额外鞋子。

【短手短腿比例冻结】
手臂和腿必须保持上传图片里的原始长度和粗细，任何一帧都不得变长。小电阻只有短而粗的卡通圆柱短臂和短腿；脚几乎直接连接在蛋体底部。禁止大腿、膝盖、小腿、长胳膊、长腿、手肘拉长、人类式跨步、人类式跑步和高抬腿。可见腿长绝不能超过上传图片中的腿长，脚底不能远离蛋体。角色移动只能通过整个蛋体轻微向前平移和极小幅度重心变化完成，不能用伸长腿、迈大步或奔跑完成。

【两只手的唯一任务与连续性】
右侧只有原图中的1条右臂和1只右手。右手从第0帧到最后一帧始终握住同一个银灰色喇叭，位置和长度基本不变；喇叭遮挡右手时也绝不能补画另一只手。
左侧只有原图中的1条左臂和1只左手。0—2.5秒，原来的左手保持抓住黑幕，不做其他手势。2.5—3.2秒，同一只左手先完全松开黑幕并短暂停顿；抓幕手完全消失后，原来的同一条左臂才开始小幅旋转。3.2—6秒，同一只左手逐渐变为面向观众的停止手势。任何一帧都不能同时出现“抓幕左手”和“停止左手”，不能为了补动作生成第三只手。

角色不走路、不跑步、不跨步，只让整个蛋体从裂缝中轻微向前平移。两条短腿保持原图长度，只做极小幅度摆动；两只脚始终分别位于蛋体左下方和右下方。唯一小嘴自然开合，对应后期口播：“等一下，等一下！我有件重要的事情要说。”灰色腰带和锯齿纹绝不张嘴。黑幕与金色裂缝保持稳定，不出现球场和其他人物。无文字、无字幕、无Logo、无水印。

宁可让角色几乎不移动，也不能改变上传图片中的形象和比例。输出前逐项核验：外形与参考图一致；1颗蛋、1圈腰带、2臂、2手、2条短腿、2只脚、1个喇叭；没有第三只手；没有大腿、膝盖、小腿和长腿。
~~~

## 生成失败时的正确处理

- 如果第一段仍出现第三只手，不再继续叠加负面提示词；重做 `01-curtain-slit-emerge.png`，让左手从首帧起就保持停止手势，不再抓幕。
- 如果腿仍被拉长，把“挤出、走出、跑、跨步”全部删除，只让整个蛋体轻微平移。
- 若首尾帧本身的手部任务不同，优先增加中间帧并拆段；不要让模型在一个片段里同时解决抓幕、挥手、持喇叭和跳跃。
