## 本集目标

- 面向谁：刷到《功夫女足》和“只能帮你到这了”宣传梗的抖音用户。
- 解决什么问题：借热点测试账号 IP 的即时反应能力，同时避免照搬热门海报和未经核验的信息。
- 观众看完得到什么：10 秒内看懂“小电阻临时当宣传广播员”的反差笑点。
- 与账号主线的关系：属于轻短梗。角色明知能力有限仍认真行动，延续“少内耗、先做点什么”的账号气质。

## 核心创意

金色夕阳球场里，卡通化的周星驰以沉默的场边导演形象出现。白色小电阻人站上木箱，抱着扩音喇叭准备帮忙宣传。它按下电源后，全身蓝色电路亮起，广播波扩散全场，远景女足球员完成凌空射门。最后角色用一本正经又有点累的语气收尾：“宣传到这了，消息等官宣。”

本集只借公共热点和网络通用句式，不复刻用户提供海报的真人背影、照片拼贴、演员形象、排版和完整文案。

## 风险与处理

| 风险 | 处理 |
|---|---|
| 热门海报来源和授权不明 | 不把原海报归档为作品素材，不复制原人物、构图和照片拼贴 |
| AI 画面可能让人误以为周星驰真人出镜 | 使用明显卡通化形象；配文明确“AI 创意呈现，并非真人出镜或真实发言” |
| 动态票房会继续变化 | 正文若使用数字必须写明“截至 2026-07-22”；封面不写票房 |
| “续集确认”缺少正式片方信息 | 本集不使用该说法，收尾明确“消息等官宣” |
| 电阻腰带容易被模型当成第二张嘴 | 所有提示词锁定唯一小嘴在眼睛与腰带之间，禁止腰带开合和牙齿化 |
| 10 秒长生成容易漂移 | 使用首尾帧、固定机位、弱运镜；失败时拆成两个 5 秒片段 |
| 蓝色电流符号被实体化成第二颗头 | 明确它是无厚度二维折线贴纸；禁止球体、头部、五官和立体化 |
| 角色被拆成上下两颗蛋、重复腰带或双脚错层 | 所有独立提示词逐段重复单蛋主体和固定四肢拓扑；腰带只是一条环绕同一蛋体的结构带；落地时左右脚必须同高、同向、同一支撑面 |
| 喇叭被吞掉或突然出现 | 锁定同一喇叭从第 0 帧到末帧持续存在，双手始终与其接触 |
| 口播时嘴部不动 | 逐时间段规定唯一小嘴的开合次数、大小、位置和闭合状态 |
