# 剪辑与声音

## 时间线

| 时间 | 画面 | 口播 |
|---|---|---|
| 0—1.5 秒 | 克制快速敲击，突然喊住观众 | 爸爸们先别划！ |
| 1.5—3.5 秒 | 手速、眼神和汗珠逐渐升级 | 我正给自己扣一呢！ |
| 3.5—6 秒 | 疯狂敲击、电光、键盘震动 | 等我把算法敲醒了，你们再走！ |

## 声音

- 0 秒直接进人声，不留空白。
- 键盘声从中速连续加快，结尾最密，但音量低于口播。
- 3.5 秒后加入少量电流“滋滋”声，不使用爆炸音。
- 角色不做口型，口播使用画外音；可使用略年轻、急促、认真到荒唐的男声或中性声。
- 最后 0.3 秒只留疯狂键盘声，便于形成笑点和循环感。

## 字幕

字幕按三句拆开，不要整段一次出现。第一句可以放大约 15%，其余保持统一字号。
