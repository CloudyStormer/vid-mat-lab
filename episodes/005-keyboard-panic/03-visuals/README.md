# 首尾帧

| 顺序 | 文件 | 画面任务 |
|---:|---|---|
| 1 | [01-keyboard-controlled-start.png](01-keyboard-controlled-start.png) | 半眯眼、略焦急但仍然克制地快速敲键盘 |
| 2 | [02-keyboard-frantic-end.png](02-keyboard-frantic-end.png) | 同机位、同场景，表情和手速升级到疯狂状态 |

[contact-sheet.jpg](contact-sheet.jpg) 用于检查首尾帧的机位、桌面、角色和灯光是否一致。

## 使用方法

1. 在即梦或可灵选择“首尾帧生成视频”。
2. 首帧上传 `01-keyboard-controlled-start.png`。
3. 尾帧上传 `02-keyboard-frantic-end.png`。
4. 设置横版 16:9、约 6 秒、固定镜头，粘贴本集视频提示词。

两张图均为 1672×941，无文字、Logo 或水印；最终建议以 1920×1080 导出。
