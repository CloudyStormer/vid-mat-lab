## 本集目标

- 面向谁：刚开始做内容、会盯着流量和互动反复焦虑的人。
- 解决什么问题：用“自己给自己扣 1”的荒诞动作，把新号求互动的窘迫拍成轻短梗。
- 观众看完得到什么：前两秒被喊住，随后通过表情和敲键盘升级获得一个迅速可懂的笑点。
- 与账号主线的关系：属于创作内耗轻短梗，延续第 002、004 集的新号数据焦虑支线。

## 核心创意

白色小电阻人为了留住观众不停敲键盘，从克制专注逐渐敲到满头大汗、双眼失控。角色一本正经地声称自己正在“给自己扣 1”，越认真越好笑。

## 成片结构

1. 前两秒突然喊：“爸爸们先别划！”
2. 角色仍较克制地快速敲键盘：“我正给自己扣一呢！”
3. 手速和表情不断升级：“等我把算法敲醒了，你们再走！”
4. 最后约 0.3 秒保持疯狂敲击，不停手。

## 遇到的问题与解决办法

| 问题 | 原因 | 解决办法 | 下次预防 |
|---|---|---|---|
| 疯狂打字容易生成四只手 | 模型会把手部残影误认为额外肢体 | 锁定两条手臂和两只手；主要用键盘震动、电光和身体抖动表现速度 | 提示词明确禁止手臂残影、重复手掌和漂浮手指 |
| 电阻腰带容易被当成第二张嘴 | 白色锯齿纹类似牙齿 | 唯一嘴必须位于眼睛与腰带之间；腰带保持刚性、禁止开合 | 所有图像和视频提示词重复同一组身份约束 |
| 求互动台词容易显得油腻 | 单纯“别走、扣 1”缺少自嘲 | 改为角色“给自己扣 1、把算法敲醒”，让求互动成为荒诞笑点 | 保持一本正经的语气，不真哭喊、不强迫观众互动 |
