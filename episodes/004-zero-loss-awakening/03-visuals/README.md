## 首尾帧使用方法

1. 在即梦或可灵选择“首尾帧生成视频”。
2. 首帧上传 `04-no-traffic-planet-start.png`。
3. 尾帧上传 `05-no-traffic-planet-end.png`。
4. 比例选 16:9，时长选 5 秒，粘贴本集视频提示词。

尾帧已经把星球替换为数据云，因此模型只需要补出扫描、轻点和“噗”开的连续过程，能明显降低角色漂移。
