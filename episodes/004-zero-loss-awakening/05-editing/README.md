# 五秒剪辑方案

## 时间线

| 时间 | 画面 | 声音 |
|---|---|---|
| 0.0—0.7 秒 | 星球扫描环转动，弱信号闪烁 | 低频扫描“滴”声 |
| 0.7—2.8 秒 | 角色冷静看着星球，只眨眼或挑眉一次 | “不行，这颗星球没流量。” |
| 2.8—3.5 秒 | 右手轻点星球 | “炸了算了。”＋轻触“咔”声 |
| 3.5—4.4 秒 | 星球变成蓝白数据云 | 小而干脆的电子“噗”声 |
| 4.4—5.0 秒 | 像素飘散，角色极轻耸肩 | 保留约半秒安静，托住笑点 |

## 剪映操作

1. 导入即梦或可灵生成的五秒视频。
2. 单独生成完整画外音：“不行，这颗星球没流量。炸了算了。”
3. 将第一句末尾对齐约 2.8 秒，第二句末尾对齐约 3.5 秒。
4. 不使用自动数字人口型，不让角色嘴部跟随台词。
5. 加三种短音效：扫描、轻触、数据“噗”；背景音乐可省略。
6. 字幕最多两行，放在下方安全区；如果画面已经足够清楚，也可只保留自动字幕。
7. 以 1920×1080、30 fps 导出。
