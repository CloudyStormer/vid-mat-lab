# 五秒图生视频提示词 v1

## 使用设置

- 模式：首尾帧生成视频。
- 首帧：[04-no-traffic-planet-start.png](../../../assets/brand/transformations/zero-loss-awakening-v1/04-no-traffic-planet-start.png)
- 尾帧：[05-no-traffic-planet-end.png](../../../assets/brand/transformations/zero-loss-awakening-v1/05-no-traffic-planet-end.png)
- 比例：16:9。
- 时长：5 秒。
- 运镜：固定镜头。
- 动作幅度：低。
- 提示词相关度：高。
- 创意度：中低。
- 自动口播与字幕：关闭。

## 完整提示词

```text
使用上传的首帧和尾帧，生成一段5秒、横版16:9、固定镜头、电影级半写实3D动画。

0—0.7秒：同一个白色圆滚滚小电阻人站在未来控制舱，左手托着一颗微型蓝色全息星球。蓝色扫描环缓慢转动一圈，星球旁边的弱信号格轻微闪烁。角色低头看着星球，表情冷静、嫌弃，但身体基本不动。

0.7—2.8秒：保持同一机位和同一角色，星球继续微弱闪烁。使用画外系统音说：“不行，这颗星球没流量。”角色只做一次很轻的眨眼或挑眉，不做口型。

2.8—3.5秒：角色用右手食指轻轻点一下星球，画外系统音接着说：“炸了算了。”

3.5—4.4秒：微型星球发出一声轻巧的“噗”，只在手掌上方变成小团蓝白色数据云、像素颗粒和几片全息轨道碎片。效果可爱、克制、搞笑，像删除一个失败的虚拟星球；不是灾难爆炸，不出现火焰和浓烟。

4.4—5.0秒：蓝色像素慢慢飘散。角色保持一脸平静，轻轻耸一下肩，停顿半秒形成笑点。

全程必须锁定角色：始终只有一个角色、两只眼睛、一张小黑嘴、两只手、两条腿；小黑嘴只能位于眼睛下方、灰色腰带上方。灰色电阻腰带和白色锯齿电阻纹绝对不是嘴，禁止开合，禁止出现第二张嘴、牙齿或舌头。保持白色圆形身体、灰色电阻腰带、白色锯齿纹、蓝色电路纹、黄色闪电核心和蓝色分段光环。禁止换人、变脸、多余肢体、多余手指、巨大火球、真实爆炸、城市毁灭、剧烈震动、快速运镜、切镜、字幕、文字、Logo和水印。
```

## 如果模型仍然乱动

将提示词末尾再追加：

```text
角色姿势和镜头几乎完全固定，只有扫描环、一次轻触、星球数据化、一次眨眼和极轻耸肩。星球是全片唯一发生明显变化的物体。
```
