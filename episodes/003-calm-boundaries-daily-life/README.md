## 本集目标

- 面向谁：在职场、消费和人情往来中经常不好意思拒绝的年轻人。
- 解决什么问题：把“拒绝内耗”从口号变成五种可以直接说出口的边界表达。
- 观众看完得到什么：轻微爽感、可模仿的表达方式，以及“不必把拒绝解释到所有人满意”的松弛感。
- 与账号主线的关系：直接承接年轻打工人的工作、消费、金钱和关系压力。

## 核心判断

真正减少内耗，不是靠吵赢别人，而是在事情发生时把需求、范围和退出条件说清楚。

## 内容结构

- 开场：明明不愿意，却还在想怎样拒绝才不会得罪人。
- 五个原创场景：职场临时加活、伴侣相处与休息需求、聚餐不合理平摊、租房签约临时加费、网约车要求线下交易。
- 结尾：不追求“怼赢”，只强调当场说清边界，避免事后反复生气。
- 视觉计划：五个场景各使用一张起始帧和一张收尾帧，共十张横版 16:9 图片；图片本身不生成文字。
