## 本集目标

- 面向谁：在职场、消费和人情往来中经常不好意思拒绝的年轻人。
- 解决什么问题：不只表达“我不接受”，而是识别对方话里的逻辑漏洞，把不该由自己承担的任务、成本和决定原路还回去。
- 观众看完得到什么：具有反拨感的生活爽点、可模仿的短句，以及“不必替不合理要求找理由”的松弛感。
- 与账号主线的关系：直接承接年轻打工人的工作、消费、金钱和关系压力。

## 核心判断

真正减少内耗，不是靠吵赢别人，也不是单纯不听，而是顺着不合理的前提追问一步，让该安排任务、支付成本或作出决定的人承担自己的部分。

## 内容结构

- 开场：很多要求看似无法拒绝，其实是对方的逻辑经不起继续往下说。
- 八个场景：深夜加班比较、团结与待遇、亲戚问工资、香火催婚、购物道德绑架、借钱不还、吃亏是福、用“说话直”包装冒犯。
- 结尾：不靠忍耐或翻脸，让该做决定的人自己作出决定。
- 视觉交付：八个场景各使用一张起始帧和一张收尾帧，共十六张横版近 16:9 图片；图片本身没有文字。
