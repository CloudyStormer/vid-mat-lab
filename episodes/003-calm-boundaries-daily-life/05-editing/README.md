# 第 003 集剪辑规划

- 当前预计总时长：约 80 至 100 秒，最终以八组配音长度为准。
- 每个场景使用一组横版起始帧和收尾帧生成 10 秒动态；若平台只有 5 秒，则在剪映延长两端静帧。
- 每段由配角先说、小电阻人后回应。配音在剪映分成两个独立音频片段，分别对准两人的口型阶段。
- 每场回应结束后在收尾帧停留 0.8 至 1.2 秒，保留反拨后的喜剧停顿。
- 转场统一使用 0.2 至 0.3 秒叠化；不要使用旋转、翻页或强闪白转场。
- 画面可做不超过 3% 的轻微推近，避免首尾帧插值时角色结构漂移。
- 结尾口播继续使用第 08 组收尾帧，不额外增加图片。
- 首尾帧之间只安排完成故事所必需的动作；非说话的背景人物默认完全静止，避免模型擅自生成惊吓、捂嘴、后跳和交换眼神。
- 每场在生成前先确认“故事是什么、谁先说、反拨点是什么、结尾人物具体在想什么”，再写动作提示词，避免把“没想到你敢反嘴”的错愕做成害怕或面无表情。

## 推荐顺序

1. 先用第 08 组 v3“没想到你敢反嘴”尾帧测试图生视频，同时检查过渡与单嘴结构。
2. 通过后按 01 至 08 依次生成动态。
3. 逐组生成两段配音，并按提示词中的说话阶段对齐。
4. 对齐画面、语音和字幕后，再加轻推近、转场和音乐。

## 必须弃用的错误片段

- 腰带开合或锯齿变成牙齿；
- 腰带区域出现粉色嘴、舌头或第二张嘴；
- 两个人同时说话或口型错位；
- 路人或背景人物无缘无故受惊、后跳、捂嘴或改变表情；
- 人物、手脚、文件和道具凭空增加；
- 场景、服装或机位中途改变。
