# 第 003 集图生视频提示词 v1｜首尾帧单嘴收紧版

> 历史版本：场景 01—07 的背景反应和首尾动作跨度经复核仍然偏大，当前请使用 [场景 01—07 v2 故事优先低动作版](scenes-01-07-video-v2-story-first.md)；场景 08 使用独立 v3。

## 使用方法

1. 在即梦或可灵选择“首尾帧 / 起始帧与结束帧”图生视频模式。
2. 每组按文件名上传对应 `start` 和 `end` 两张图。
3. 八组统一先选 **10 秒**；如果平台只有 5 秒，先生成 5 秒动作片段，再在剪映用起始帧和收尾帧各延长 1 至 2 秒承接口播。
4. 不让生成工具自动写字幕，不使用自动生成的画面文字；口播和字幕在剪映后期添加。
5. 每段先由人类配角说话，再由小电阻人回应；禁止两个人同时做口型。
6. 场景 08 已由单独的 [v3“没想到你敢反嘴”方案](scene-08-video-v3-incredulous.md) 取代；先测试 v3，确认轻微错愕、过渡自然且腰带没有张合后，再修订其余七组。

## 出错判定

出现下列任意一种情况，整段弃用并从两张原始静帧重新生成，不要拿错误视频截图继续扩展：
