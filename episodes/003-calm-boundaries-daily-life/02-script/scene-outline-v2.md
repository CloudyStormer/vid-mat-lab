# 第 003 集场景与对白 v2｜逻辑反拨版｜待确认

## 暂定标题

《不吵架，把问题原路还回去》

## 本轮为什么重写

v1 的主角主要在表达“我不接受”，但对方提出的不合理前提没有被拆穿，冲突也结束得太快，因此只有边界感，没有短剧需要的反拨感和记忆点。

v2 统一使用四拍结构：

1. 对方先抛出一句听起来理所当然、实际站不住的话。
2. 小电阻人不争吵，顺着对方的前提再往下推一步。
3. 对方必须面对自己话里的漏洞，并亲自作出选择。
4. 收尾帧出现明确的结果动作，不能只靠主角转身离开。

表演语气平静、短促，关键反问前停顿半拍。爽点来自逻辑闭环，不来自提高音量或羞辱对方。

## 形式

- 预计时长：约 60 至 75 秒，以最终配音为准。
- 画幅：横版 16:9，建议 1920×1080。
- 主角：固定白色小电阻人。
- 场景数量：5 个。
- 图片数量：每场 1 张起始帧 + 1 张收尾帧，共 10 张。
- 图片不生成文字；金额、项目名称和对白全部在配音与字幕阶段添加。

## 开场口播

```text
我后来发现，很多事不是没法拒绝，是对方那套话，根本经不起顺着往下说。
```

开场画面不单独生成，使用五个场景起始帧的快速闪切。

## 场景 1｜三个任务都是“第一优先级”

### 起始帧

临近下班的办公室。小电阻人桌上并排放着三个颜色不同的厚文件夹，主管用手同时压住三个文件夹，神情急切，示意全部今晚完成。

### 对白

主管：

```text
这三件今晚都要，都是第一优先级。
```

小电阻人：

```text
三个都第一，就等于没第一。您挑一个我今晚做，剩下两个明天排。
```

### 收尾帧

主管停顿后，把其中两个文件夹放进旁边的待办架，只留下一个在小电阻人桌面。小电阻人坐下打开电脑，开始处理唯一确定的任务。

### 反拨点

主角没有拒绝工作，而是把“所有事情都很急”的管理问题还给负责排优先级的人。

### 收尾字幕

```text
什么都最急，就等于没有优先级。
```

## 场景 2｜“一起做的”，那就一起讲清分工

### 起始帧

会议开始前，同事把小电阻人制作的方案册拿到自己面前，同时伸手去拿演示遥控器。小电阻人的桌面还放着草稿、素材和修改记录。

### 对白

同事：

```text
这套方案我来讲吧，反正是咱俩一起做的。
```

小电阻人：

```text
好啊。你讲你做的部分，我讲我做的部分，正好把分工说清楚。
```

### 收尾帧

会议现场，同事动作僵住后，把演示遥控器交给小电阻人。主管抬手示意小电阻人继续，方案册和草稿同时摆在两人面前。

### 反拨点

主角接受“一起做”的说法，再顺势要求按真实分工展示，让抢功的人无法只拿“一起”的好处。

### 收尾字幕

```text
成果一起讲，分工也一起讲。
```

## 场景 3｜“大家不会差你钱”，那就现在转

### 起始帧

朋友聚在咖啡桌旁讨论旅行。几个人把酒店和车票页面都推到小电阻人面前，示意它先统一付款；小电阻人拿着银行卡，其他人的手机还放在桌上没有动作。

### 对白

朋友：

```text
酒店和车票你先垫一下呗，大家又不会差你这点钱。
```

小电阻人：

```text
既然不会差，那现在转和回来转没区别。钱齐了，我马上订。
```

### 收尾帧

刚才催促的朋友第一个拿起手机付款，其他人也跟着操作。等大家全部转完，小电阻人才把银行卡放到预订页面旁边准备下单。

### 反拨点

“肯定会还”被顺着推成“那就现在付”，空头承诺当场变成实际动作。

### 收尾字幕

```text
既然不会差，那就不用等回来。
```

---

## 场景 4｜“在乎我”不能只要求一个人证明

### 起始帧

周末家中。明显疲惫的小电阻人裹着薄毯坐在沙发上，桌上放着水杯和纸巾；伴侣已经穿好外套、拎着购物袋，站在门边要求它一起出门。

### 对白

伴侣：

```text
你要是真的在乎我，今天就应该陪我出去。
```

小电阻人：

```text
那你也在乎我，是不是也该让我休息？今天一起在家，还是明天下午一起出去，你选。
```

### 收尾帧

伴侣停顿后放下外套和购物袋，拿来一条毯子坐到沙发另一边。两个人一起喝热水，原本紧绷的气氛缓下来。

### 反拨点

把单向的“爱我就证明”推回到“在乎应该相互”，同时给出两个可执行的选择，不把关系推向冷战。

### 收尾字幕

```text
在乎是相互的，不是单向证明。
```

## 场景 5｜拆开检测，不等于同意维修

### 起始帧

电子维修店柜台。维修人员把已经拆开的笔记本电脑和一个价格很高的新零件摆在小电阻人面前，用手指着拆开的机器制造“已经做到这一步”的压力。

### 对白

维修人员：

```text
机器都拆开了，现在不修多亏。换这个八百。
```

小电阻人：

```text
约定的检测费我照付，维修我没同意。您是现在装回去，还是给我完整报价，等我确认后再修？
```

### 收尾帧

维修人员把零件重新装回机器，将装好的电脑和完整报价单一起推回。小电阻人支付约定的检测费，收好机器，没有被沉没成本逼着现场决定。

### 反拨点

对方把“已经检测”偷换成“必须维修”，主角把检测和维修重新拆开，也保留了对合理检测费的承担。

### 收尾字幕

```text
做过检测，不等于同意维修。
```

## 结尾口播

```text
真正的不内耗，不是忍着不听，也不是当场翻脸。是把对方的逻辑推回去，让该做决定的人，自己做决定。
```

结尾继续使用场景 5 的收尾帧，镜头缓慢拉远，不额外增加图片。

## 生成图片前的确认项

1. 五个场景是否都保留。
2. 每场的反问是否够自然、够有劲，但没有变成刻意抬杠。
3. 是否需要把某一场替换为“亲戚催婚买房”或“统一随礼”场景。
4. 用户确认后再生成 10 张起始帧与收尾帧；本版本仍不生成图片。
